# Flask Domain Setup 🚀

This repository contains a **Bash script to connect your Flask app to a domain**, configure **Nginx**, run your app in **tmux**, and enable **HTTPS via Let’s Encrypt**. Perfect for hosting multiple Flask apps on one VPS.
